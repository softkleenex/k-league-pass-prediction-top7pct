2. 주요 기능
================================================================================

analyze.py 주요 기능:
  1. CV 결과 로드 (cv_results.json)
  2. CV 비교 분석 (기존 vs Phase 1-A)
  3. 신규 피처 분석 (5개 피처 설명 및 중요도)
  4. 통계적 유의성 검증 (95% 신뢰도)
  5. 제출 결정 기준 평가 (4가지 기준)
  6. 종합 분석 보고서 생성 (JSON)
  7. 마크다운 비교표 생성 (ANALYSIS.md)

제출 결정 기준:
  - CV < 15.5: 🚀 강력 추천
  - CV 15.5-15.6: ✅ 조건부 추천
  - CV > 15.6: ⚠️ 재검토 필요

5. 신규 5개 피처
================================================================================

1. is_final_team (⭐⭐⭐⭐⭐)
   - 골 넣은 팀의 패스 여부 플래그
   - 공격/수비 맥락 구분
   - 기대효과: 0.05-0.10점 개선

2. team_possession_pct (⭐⭐⭐⭐)
   - 최근 20개 패스 중 우리 팀 점유율
   - 조직적 공격 vs 역습 구분
   - 기대효과: 0.03-0.06점 개선

3. team_switches (⭐⭐⭐)
   - 공수 전환 누적 횟수
   - 경기 혼란도 / 템포 파악
   - 기대효과: 0.02-0.04점 개선

4. game_clock_min (⭐⭐⭐)
   - 경기 시작부터 경과 시간 (0-90분+)
   - 전반/후반 구분 제거
   - 기대효과: 0.01-0.03점 개선

5. final_poss_len (⭐⭐)
   - 현재 연속 우리 팀 소유 패스 수
   - 빌드업 vs 단발성 구분
   - 기대효과: 0.01-0.02점 개선
